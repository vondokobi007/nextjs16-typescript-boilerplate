# Next.js 16 + TypeScript Boilerplate

Production-ready starter template for Next.js 16 (App Router) with TypeScript, Tailwind CSS, ESLint, Prettier.

## Features

- Next.js 16 (App Router + Server Actions)
- TypeScript (strict mode)
- Tailwind CSS v3.4+
- ESLint + Prettier configured
- Absolute imports
- Ready for deployment (Vercel, Netlify)

## Getting Started

```bash
git clone https://github.com/vondokobi007/nextjs16-typescript-boilerplate.git
cd nextjs16-typescript-boilerplate
npm install
npm run dev
